My first website


This is the first thing I ever coded in using javascript and it was ready on 11.3. This was an assignment from course "Basics of web development". Again this is more about practising and exploring javascript so I didn't spend any extra time to make it responsive/graphically super beutiful. In process I got familiar with using API's, JSON, jQuery and Javascript. I used OpenWeatherMap's API, JSON file of largest cities in world and jQuery AutoComplete plugin to make it easy to check weather almost everywhere in world.

Goal

Creating a media element with chancing content and practice using javascript and open API in the process. As result one needed to have a a weather information service with option to choose the location and a chance to enable automatic update for the information.

In a nutshell

  • February 2019

  • Practicing Javascript

  • OpenWeatherMap API

  • Jquery autocomplete plugin

  • JSON file of largest cities in world

Result

Website where you can check current weather and the result is displayed by icon based on weather description and some numeric values ( temperature, wind, humidity). Weather statistics are from OpenWeatherMap's API. The icons are from Erikflowers Weather Icon Github repository, because OpenWeatherMap's icons had too bad quality. As bonus I used Jquery autocomplete- plugin and JSON file containing over 80 000 largest cities in different countrys.
